* Overview

This document lists custom keybindings that use the =s-= prefix (macOS
Command key) from the configuration.

Source of truth: [[file+emacs:/Users/jay/emacs/emacs-settings/keys.el][keys.el]].

For the full keybinding reference (all keys, all maps, with annotations),
see [[file+emacs:/Users/jay/emacs/emacs-settings/docs/keybinding-reference.org][keybinding-reference.org]].

Notes:
- This document does *not* include default Spacemacs bindings.
- Some bindings are mode-specific overrides (see the org-mode section at the end).
